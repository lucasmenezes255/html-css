O HTML5 é focado exclusivamente na SEMÂNTICA do código, na estrutura significativa dele. tags como:
center
bgcolor
u
b
blink
marquee

Podem funcionar, mas não são indicadas de usar no arquivo HTML. Tags de formas e estilos devem ser feitas no CSS. No HTML ainda dá pra usar cores usando a tag style na HEAD.

Para mais informações de tags obsoletas no HTML5, checa o site: https://www.w3.org/TR/2014/REC-html5-20141028/obsolete.html
